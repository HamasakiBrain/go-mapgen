# mapgen

Анализатор Go-проектов на базе **go/types + SSA/CHA**: строит точный граф
вызовов (методы, интерфейсы, кросс-пакетная и динамическая диспетчеризация
разрешаются по типам, а не по именам), считает реальные метрики сложности,
находит HTTP-маршруты и циклические импорты, отвечает на запросы
**impact** («что сломается, если изменить X») и **trace** («путь вызова от A к B»),
и рендерит интерактивный HTML-дашборд.

## Требования

- Go **1.25+** (если версия старее — Go сам подтянет нужный toolchain при сборке).
- Зависимости уже завендорены в `vendor/`, интернет для сборки не нужен.

## Сборка

```bash
cd mapgen
go build -o mapgen .
```

Получится один бинарник `mapgen`. Можно положить в PATH или вызывать локально.

## Использование

Запускать на **корне любого Go-проекта** (там, где `go.mod`):

```bash
# Сгенерировать все артефакты (json, md, html, dot) с префиксом mapgen-*
./mapgen -path /путь/к/проекту -out отчёт

# Открыть интерактивный дашборд в браузере
./mapgen -path /путь/к/проекту -web        # http://localhost:8080

# Импакт-анализ: что зависит от функции/метода
./mapgen -path /путь/к/проекту -impact ИмяФункции -depth 5

# Трассировка пути вызова
./mapgen -path /путь/к/проекту -trace 'From..To'
```

### Флаги

| Флаг | По умолчанию | Описание |
|------|--------------|----------|
| `-path` | `.` | корень анализируемого проекта |
| `-out` | `mapgen` | префикс выходных файлов |
| `-format` | `all` | `json,markdown,html,dot` или `all` |
| `-impact` | — | символ для импакт-анализа |
| `-depth` | `4` | глубина обхода для `-impact` |
| `-trace` | — | путь вызова в формате `<from>..<to>` |
| `-web` | `false` | поднять дашборд |
| `-port` | `8080` | порт дашборда |

Символы в `-impact`/`-trace` можно указывать коротко (`Save`), как `Тип.Метод`
(`MemStore.Save`) или полным ID (`example.com/pkg.(MemStore).Save`).

## Дашборд (HTML)

`mapgen.html` — самодостаточная страница (D3 подгружается с CDN, нужен интернет
при открытии): карточки метрик, граф зависимостей пакетов (тянется мышью),
проблемы, маршруты и поиск/сортировка по всем функциям.
